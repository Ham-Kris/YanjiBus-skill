# 延吉公交实时查询

当用户想查询延吉公交线路的实时车辆位置时，使用此 skill。

## 触发条件
用户提到查公交、延吉公交、几路车、公交到哪了、等公交等相关内容时触发。

## 使用方法

运行查询脚本：
```bash
bash ./yanji-bus.sh <线路号> [子线路号]
```

## 输出解读

- **stationnum**: 车辆当前所在的站序号（从1开始）
- **speed**: 车速 km/h
- **updown**: 0=上行, 1=下行
- **longitude/latitude**: 车辆经纬度

## 注意事项

- 某些线路有多条子线路（如3路有3条线），需指定子线路号
- 如果用户只说了线路号没说子线路，默认用子线路1
- 数据来源为延吉公交集团微信公众号接口，仅供个人查询使用

## 配合高德地图 skill 使用

可搭配 [高德地图 skill](https://github.com/kaichen/amap-skill) 实现更完整的出行体验：
- 根据目的地名称查找最近的公交站点
- 从当前位置步行导航到公交站
- 下车后步行导航到最终目的地
- POI 搜索确认目的地位置
